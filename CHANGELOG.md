# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure
- Neo-retro theme with CRT effects
- Dashboard layout with command menu sidebar
- Quest management system
- Inventory management system
- Interactive map interface
- Settings configuration panel
- Supabase integration utilities
- Reusable UI components (GlitchText, PixelButton, GlowCard)
- Docker support for development and production
- Comprehensive documentation

### Changed
- Improved responsive design for mobile devices
- Enhanced animation performance with Framer Motion
- Optimized Tailwind CSS configuration

### Fixed
- Minor CSS glitches in the CRT effect
- Responsive issues in the command menu
- Typing animation timing adjustments

## [1.0.0] - 2026-03-29

### Added
- Initial release of Neo-Retro Dashboard
- Complete 90s point-and-click adventure UI
- Dungeon crawler inspired interface
- Full Next.js 15 App Router implementation
- TypeScript type safety
- Tailwind CSS styling
- Framer Motion animations
- Supabase backend integration (mock implementation)
- Responsive design for all screen sizes
- Dark theme with poison green and plasma blue accents
- Glitch effects and CRT scanline overlay
- Pixel-perfect button interactions
- Comprehensive documentation and deployment guides
