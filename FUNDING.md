# Funding

Thank you for your interest in supporting the Neo-Retro Dashboard project! While this is primarily a volunteer effort, we appreciate any contributions that help us maintain and improve the project.

## Ways to Support

### Code Contributions
The most valuable contribution you can make is helping us improve the codebase:
- Fix bugs and submit pull requests
- Implement new features
- Improve documentation
- Help with testing and quality assurance

### Community Support
- Answer questions in discussions
- Help other users troubleshoot issues
- Share the project with others
- Write tutorials or blog posts

### Financial Support
While we don't currently have a formal sponsorship program, you can support the maintainers through:

#### GitHub Sponsors
